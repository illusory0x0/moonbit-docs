# 块和语句

块是一个包含一系列语句和一个可选表达式的表达式。

```
{
  statement1
  statement2
  expression
}
```

例如，上面的代码将按顺序执行 `statement1`、`statement2` 和计算 `expression`。块的计算结果是表达式的计算结果。如果省略最后一个表达式，
块将返回 `()`，其类型为 `Unit`。

语句可以是变量声明、变量赋值或任何类型为 `Unit` 的表达式。

块还与命名空间作用域相关联。在 `main` 子句中，内部块中声明的变量 `a` 将遮蔽外部的 `a`。它仅在内部块中可见。
